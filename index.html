FRONT-END : Front-end means the front side of a website, we can call it a user interface. Contains : texts , buttons,images,videos,tags etc,all these are used for making the interaction with the website cool.


What is Frontend ?
HTML,CSS and Javascript are the main building blocks for creating ant website. "A front-end developer plays a vital role in designing and implementing an application's user interface, which serves as the primary point of entry for users to access and interact with the application. While a web designer is responsible for shaping the visual appearance and layout of a website, the front-end developer ensures that this design is translated into a functional online experience by utilizing coding languages like CSS, HTML, and JavaScript to bring the design to life."
